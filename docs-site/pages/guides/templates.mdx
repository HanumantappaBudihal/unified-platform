# App Templates

Scaffold a new application with pre-configured platform integrations using the CLI.

## Available Templates

| Template | Stack | Includes |
|----------|-------|----------|
| `node-api` | Express + PG + Redis + Kafka | REST API, health check, Docker Compose |
| `next-app` | Next.js + NextAuth + MinIO | SSO, file uploads, Docker Compose |

## Usage

```bash
platform init my-service --template node-api
cd my-service
npm install
```

## Template Features

All templates include:
- **Dockerfile** with multi-stage build and health check
- **docker-compose.yml** that joins platform networks
- **.env.example** with all platform connection variables
- **/health** endpoint for readiness probes
- Pre-configured to work with `platform onboard` credentials

## After Scaffolding

1. Run `platform onboard` to provision infrastructure
2. Copy credentials from `platform status --credentials` to `.env`
3. Run `npm run dev` to start development
4. Run `docker compose up` to deploy on the platform network
