# Infrastructure Platform

A complete, self-hosted infrastructure layer for running applications with multi-environment support and enterprise-grade security.

## What's Included

| Layer | Services |
|-------|----------|
| **Data** | PostgreSQL, Redis Cluster, Kafka (KRaft), MinIO |
| **Security** | OPA (Operator Authorization), Kong (API Gateway), TLS |
| **Observability** | Prometheus, Grafana, Loki, Alertmanager, Uptime Kuma |
| **Platform** | Unified Portal, Backup Server, Email Server |

## Quick Start

```bash
# 1. Generate secrets
./scripts/generate-secrets.sh

# 2. Start infrastructure
cd database-server && docker compose up -d
cd ../cache-server && docker compose up -d
cd ../event-streaming-server && docker compose up -d
cd ../object-storage-server && docker compose up -d
cd ../authz-server && docker compose up -d
cd ../api-gateway-server && docker compose up -d
cd ../unified-gateway-portal && docker compose up -d

# 3. Open the portal
open http://localhost:3006
```
